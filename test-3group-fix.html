3组qPCR功能修复测试

修复内容：

  1. 修正了 generate3GroupDataRows 方法中的数据访问路径
  2. 修正了 generateAll3GroupsDataRows 方法中的数据访问路径
  3. 原错误：尝试访问 comparison.rawData.control.refCt
  4. 修正后：正确访问 comparison.rawData.controlRefCt

错误原因分析

performQPCRCalculation 方法返回的 rawData 结构是：

    rawData: {
        controlRefCt,    // 对照组参考基因Ct值数组
        controlTargetCt, // 对照组目标基因Ct值数组
        expRefCt,        // 实验组参考基因Ct值数组
        expTargetCt      // 实验组目标基因Ct值数组
    }

而不是：

    rawData: {
        control: { refCt: [], targetCt: [] },
        experimental: { refCt: [], targetCt: [] }
    }

测试步骤

  1. 打开实验管理系统
  2. 进入"数据分析" → "qPCR分析"
  3. 选择"文件导入"标签
  4. 选择含有3组数据的qPCR文件
  5. 选择"3组两两比较"
  6. 选择参考基因、目标基因和3个组
  7. 点击"开始分析"
  8. 验证结果显示完整的Ct值数据表

返回实验管理系统
