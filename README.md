# 实验管理系统

一个专为研究生设计的本地实验管理网页应用，帮助你系统化地管理实验安排和实验结果。

## ✨ 主要功能

### 📊 实验概览
- 实时统计活跃实验、已完成实验和记录数量
- 显示最近的实验活动
- 月度数据统计

### 🧪 实验管理
- 创建、编辑、删除实验项目
- 设置实验分类、开始和结束日期
- 跟踪实验状态（进行中/已完成/计划中）

### 📝 实验记录
- 详细记录实验过程和结果
- 记录实验参数和观察结果
- 按实验、日期和关键词过滤记录

### 📈 数据分析
- 实验完成率统计
- 月度记录趋势
- 分类统计分析

### 📁 文件管理
- 上传和管理实验相关文件
- 文件下载和删除功能
- 支持多种文件格式

## 🚀 快速开始

### 方法一：Node.js 服务器（推荐）

1. **检查环境**
   - 确保已安装 [Node.js](https://nodejs.org/) (版本 14 或更高)

2. **启动应用**
   ```bash
   # Windows 用户
   双击运行 start.bat

   # 或者手动执行
   npm install
   npm start
   ```

3. **访问应用**
   - 在浏览器中打开 http://localhost:3000

### 方法二：Python 简单服务器

1. **启动服务器**
   ```bash
   # 确保已安装 Python 3
   python start.py
   ```

2. **访问应用**
   - 在浏览器中打开 http://localhost:8080

### 方法三：直接打开（功能受限）

直接双击 `index.html` 文件，但某些功能可能无法正常工作。

## 💾 数据存储

- **本地存储**：所有数据保存在浏览器的 localStorage 中
- **数据导出**：支持导出为 JSON 文件进行备份
- **数据导入**：可以导入之前导出的数据文件
- **隐私保护**：所有数据仅存储在本地，不会上传到任何服务器

## 🔧 使用说明

### 创建实验

1. 点击"实验管理"标签
2. 点击"新建实验"按钮
3. 填写实验信息（标题、描述、分类等）
4. 点击"创建实验"完成

### 添加记录

1. 点击"实验记录"标签
2. 点击"添加记录"按钮
3. 选择关联的实验项目
4. 填写记录内容、参数和结果
5. 点击"添加记录"完成

### 数据备份

1. 在任意页面，使用浏览器开发者工具
2. 在控制台执行：`app.exportData()`
3. 或者在代码中添加导出按钮

### 数据恢复

1. 准备之前导出的 JSON 文件
2. 在控制台执行：`app.importData(file)`
3. 或者在代码中添加导入功能

## 🎨 界面特色

- **现代化设计**：采用渐变色和卡片式布局
- **响应式设计**：支持桌面和移动设备
- **直观导航**：清晰的标签页导航
- **动画效果**：流畅的过渡动画
- **用户友好**：简洁的操作界面

## 🛠️ 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **后端**：Node.js + Express (可选)
- **存储**：localStorage + JSON 文件
- **服务器**：Express / Python HTTP Server

## 📋 系统要求

- **浏览器**：现代浏览器（Chrome, Firefox, Safari, Edge）
- **JavaScript**：必须启用
- **存储空间**：约 5-10MB 本地存储空间

## 🚨 注意事项

1. **数据安全**：定期导出数据进行备份
2. **浏览器限制**：清除浏览器数据会丢失所有记录
3. **文件上传**：上传的文件仅在当前会话中有效
4. **网络要求**：无需联网，完全本地运行
