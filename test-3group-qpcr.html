🧬 3组qPCR两两比较功能 - 完整使用说明

🎉 更新通知

文件导入功能现已支持3组比较！您现在可以在手动输入和文件导入两种模式下使用3组两两比较功能。

✨ 新功能概述

实验管理系统的qPCR分析模块现已完整支持3组数据的两两比较！无论是手动输入数据还是从文件导入，都可以进行3组比较分析。

📋 功能特点

  • 双模式支持：手动输入和文件导入都支持3组比较
  • 自动两两比较：系统自动进行组1 vs 组2、组1 vs 组3、组2 vs 组3的比较
  • 统计分析：每个比较都提供完整的统计数据（t检验、p值、倍数变化）
  • 可视化结果：清晰的表格和图表展示所有比较结果
  • 数据导出：支持JSON和CSV格式导出，方便后续分析
  • 结果保存：可将分析结果保存到实验记录中

🚀 使用方法一：手动输入模式

步骤1：进入qPCR分析模块

打开实验管理系统 → 点击"数据分析"标签 → 选择"qPCR分析"

步骤2：选择数据来源

确保选中"手工输入"标签

步骤3：选择分析类型

在"选择分析类型"部分，点击"3组两两比较"按钮

步骤4：输入数据

输入参考基因、目标基因名称，以及3个组的Ct值数据

🚀 使用方法二：文件导入模式

步骤1：上传qPCR数据文件

先在"文件管理"中上传包含qPCR数据的文件（支持CSV、Excel等格式）

步骤2：进入qPCR分析

点击"数据分析" → "qPCR分析" → 选择"文件导入"标签

步骤3：选择文件并配置分析
  1. 从文件列表中选择您的qPCR数据文件
  2. 在数据分析设置界面，选择"3组两两比较"
  3. 选择参考基因和目标基因
  4. 分别选择3个要比较的组
  5. 点击"开始分析"

📊 数据文件格式要求

文件应包含以下列：
- Target（或Target Name）：基因名称
- CT（或Ct、Cq）：Ct值
- Biological Set Name（或Sample）：样本组名称

示例数据：
Target,CT,Biological Set Name
GAPDH,20.1,Control
IL-6,25.1,Control
GAPDH,20.0,Treatment1
IL-6,22.1,Treatment1
GAPDH,20.2,Treatment2
IL-6,23.2,Treatment2

📈 分析结果说明

3组比较结果包含：

  1. 比较结果汇总表：展示所有两两比较的倍数变化、p值和显著性
  2. 详细统计数据：每个比较的ΔCt值、相对表达量及其标准误
  3. 可视化图表：柱状图直观展示3个组的相对表达量
  4. 统计检验：独立样本t检验结果
💡 注意事项：
  • 组1始终作为对照组，其相对表达量设为1.000
  • 每个比较中，第一个组作为对照，第二个组作为实验组
  • 显著性水平：*p<0.05, **p<0.01, ***p<0.001
  • 确保每组至少有2个重复数据以进行统计分析
  • 文件导入模式下，目前仅支持单基因的3组比较

💾 数据保存与导出

  • 保存到实验记录：点击"保存结果到实验记录"按钮，将完整的分析报告保存
  • 导出JSON：包含所有原始数据和计算结果，适合程序化处理
  • 导出CSV：表格格式，方便在Excel中进一步分析
